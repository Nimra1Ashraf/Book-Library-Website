# 📚 Personal Book Library

A simple web app to manage a personal book collection. Users can add, search, categorize, and track borrowed books easily.

## 🚀 Features

* Search by title or author
* Categorize books
* Track borrowing history

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

## ▶️ How to Use

1. Clone the repository
2. Open `index.html` in your browser
